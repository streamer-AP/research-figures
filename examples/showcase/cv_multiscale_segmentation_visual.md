# Coarse-to-Fine 3D Corridor Segmentation

Generate a paper-style method overview illustration for a computer vision and 3D perception system.

Core story:

- input a large transmission-corridor point cloud
- perform whole-scene downsampling to preserve global geometry
- run a coarse segmentation backbone over the scene
- focus on thin wires and rare components with a sparse refinement branch
- fuse coarse and refined predictions into a dense semantic map

Visual elements:

- a colored 3D point cloud corridor scene on the left
- a central multi-stage segmentation pipeline with two levels: coarse branch and refinement branch
- thin wire structures and tower components highlighted as difficult targets
- a clean segmentation result panel on the right

Style:

- white or very light background
- publication-quality CV figure
- modern scientific rendering
- limited readable labels
- emphasize coarse-to-fine reasoning rather than generic AI art
