# Scientific Figure Prompt Patterns

## Shared Rules

- white or very light background
- publication-quality composition
- minimal clutter
- very limited readable labels
- no watermark
- no paragraphs inside the image

## Method Overview

Prompt for:

- a left-to-right or layered composition
- 3-5 major stages
- one feedback loop only if important

## Visual Abstract

Prompt for:

- one central story
- balanced composition
- 3-5 scientific motifs

## Teaser

Prompt for:

- a stronger hero composition
- fewer labels
- higher visual impact

## System Concept

Prompt for:

- central system object
- environment cues only when they clarify the system
