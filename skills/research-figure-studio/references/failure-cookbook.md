# Failure Cookbook

## Too Much Text In The Image

Fix:

- reduce label set
- remove explanatory sentences
- keep only stage titles

## Structure Drift In Image Generation

Fix:

- tighten prompt
- reduce stage count
- switch to `drawio`

## Diagram Looks Too Generic

Fix:

- add domain objects
- emphasize the paper's unique mechanism
- use stronger visual anchors

## Chart Requested But Image Backend Was Chosen

Fix:

- reroute to `plot`

## Patent Figure Looks Decorative

Fix:

- stop image generation
- reroute to `drawio`
