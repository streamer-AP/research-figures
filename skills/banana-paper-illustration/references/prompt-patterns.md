# Prompt Patterns

## Shared Constraints

Use these in most prompts:

- clean academic illustration
- publication-quality composition
- white or very light background
- scientific and polished
- minimal clutter
- limited readable labels only
- no watermark
- no UI screenshot look
- no stock-photo look unless explicitly wanted

## Teaser

Use for:

- the opening figure of a paper
- a visually striking summary

Pattern:

- one dominant subject
- two or three supporting elements
- stronger atmosphere
- less diagram structure

## Method Overview

Use for:

- a high-level pipeline figure
- agentic workflows
- multi-stage method summaries

Pattern:

- structured left-to-right or layered composition
- a few major modules
- explicit feedback loop if important
- very limited text

## Visual Abstract

Use for:

- whole-paper summaries
- concise scientific communication

Pattern:

- balanced composition
- one central result or system
- 3-5 scientific motifs

## System Concept

Use for:

- hardware-software systems
- robotics, drones, multi-agent systems

Pattern:

- central system in foreground
- supporting infrastructure around it
- environment cues only if they help explain the system
