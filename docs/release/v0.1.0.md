# v0.1.0 Release Notes

This is the first GitHub-facing release package for `research-figures`.

## Highlights

- added a top-level `./research-figures` wrapper for easier onboarding
- added `./research-figures demo --offline` for first-run success without an API key
- added synchronized caption generation via `figure.caption.md`
- improved caption quality with source-aware fallback for sparse Markdown and TeX inputs
- tightened README, quickstart, and contributor docs around reproducible examples

## Suggested GitHub Release Summary

`research-figures` turns paper text, tables, and patent drafts into publication-ready figures with one entry point. This release adds an offline demo path, a top-level CLI wrapper, synchronized caption generation, and clearer docs for contributors and first-time users.
