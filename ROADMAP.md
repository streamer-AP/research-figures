# Roadmap

## Now

- sharpen first-run conversion from clone to successful demo
- keep figure artifact, caption, and verification bundle in sync
- improve offline-safe examples for drawio, plot, and hybrid
- keep Banana prompts publication-focused and domain-aware

## Next

- source-aware fallback for Banana prompt compilation, not only captions
- stronger hybrid layouts with tighter plot and structure composition
- richer caption templates for comparative plots and ablation figures
- lightweight packaging beyond repo-local wrapper scripts

## Later

- broader LaTeX figure/table ingestion
- more robust dataset-to-plot extraction helpers
- optional GitHub Action for demo regeneration on release branches
- more showcase packs for patents, robotics, and systems papers

## Explicit Non-Goals For The Near Term

- full WYSIWYG editor replacement
- real-time collaborative design tooling
- perfect topology control in image-generation backends
